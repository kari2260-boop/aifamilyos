# kari-writing-style

# Kari 写作风格与内容样本库

## 核心写作特征

### 1. 结构化思维
- **多层级标题**: 清晰的章节划分（第一部分、第二部分...）
- **视觉化呈现**: 大量使用表格、流程图、ASCII图表
- **模块化设计**: 每个部分独立完整，可单独阅读

### 2. 金句文化
- **每个关键点都有金句**: 用引用块突出核心观点
- **金句特点**: 简短有力、对比鲜明、易于传播
- **示例**:
  - "未来不是等来的，是被你做出来的。"
  - "能力不是听懂的，是做出来并被验证的。"
  - "鸡血救一天，系统救一年。"
  - "我们是小炒，不是自助餐；是手工现做的寿司。"

### 3. 对比呈现
- **表格对比**: 传统 vs 创新、问题 vs 方案
- **清晰差异化**: 突出独特价值
- **示例结构**:
  ```
  | 传统方式 | 我们的方式 |
  |---------|-----------|
  | 问题点   | 解决方案   |
  ```

### 4. 数字化表达
- **具体数据**: 80+导师、100万定价、2-3年周期
- **量化成果**: 94%录取率、效率提升30%+
- **精确时间**: 20分钟、40分钟、60分钟

### 5. 视觉符号系统
- **Emoji使用**: 🚀 🎯 ⭐ 🔍 🤖 等，增强可读性
- **颜色标记**: 🔴 🟡 🟢 🔵 用于分类和强调
- **图标系统**: ✅ ❌ ⚠️ 💡 用于状态和提示

## 典型内容模板

### 模板1: 课程/训练营设计

**结构**:
```
1. 封面/标题（带Slogan）
2. 核心目标（递进式3个目标）
3. 背景/痛点（3大危机/挑战）
4. 核心能力/方法（6宫格/多维度）
5. 角色分配（清晰定义）
6. 铁律/原则（红色警示）
7. 详细流程（时间轴+产出）
8. 关卡详解（每关独立说明）
9. 验收标准（具体可操作）
10. 金句总结
```

**特点**:
- 游戏化语言（关卡、徽章、CEO、CTO）
- 强调"可验收"、"可交付"
- 注重"系统"而非"鸡血"
- 家长角色定位为"验收官"而非"警察"

### 模板2: 商业方案/BP

**结构**:
```
1. 全球最佳实践对标
2. 产品定位（一句话+差异化）
3. 核心价值（三方融合/多维度）
4. 产品详情（定价+周期+客群）
5. 核心服务模块（支柱式）
6. 服务交付流程（阶段式）
7. 结果保障与风险管控
8. 定价逻辑与价值支撑
9. 可执行性与可持续性
10. 销售话术与价值呈现
11. 产品架构总览（ASCII图）
```

**特点**:
- 国际对标建立权威性
- 价值溢出逻辑（定价100万，价值280-510万+）
- 结果保障机制（对赌条款）
- 多方融合价值（傲客+甄爱+乐博）

### 模板3: 工具/方法卡片

**结构**:
```
┌─────────────────────────────────┐
│ 📌 主题：________________       │
│ 🔑 关键点：1._ 2._ 3._         │
│ ❓ 问题：________________       │
│ ✅ 验收：________________       │
│ ⚠️ 注意：________________       │
└─────────────────────────────────┘
```

**特点**:
- 框线设计，视觉清晰
- 填空式，可直接使用
- 结构化，易于复制

## 语言风格特征

### 1. 直击痛点
- "孩子被赋权的那一刻，成长开始发生"
- "未来最贵的不是知识，是辨别知识的能力"
- "学会停下来的人，才真正掌控学习"

### 2. 系统思维
- "三位一体"、"飞轮模型"、"四象限"
- 强调"系统"、"流程"、"SOP"
- 注重"可复用"、"可迁移"

### 3. 实战导向
- "可验收"、"可交付"、"可量化"
- "证据在哪"、"怎么证明"
- "最小可交付成果"、"MVP"

### 4. 赋能语言
- "不是...而是..."（对比句式）
- "从...到..."（转化句式）
- "让...变成..."（赋能句式）

### 5. 权威建立
- 引用国际案例（Command Education、IvyWise）
- 数据支撑（80+导师、300+案例）
- 官方背书（宾大授权、博士学位）

## 内容创作流程

### 第一步：定位与框架
1. 明确目标受众
2. 提炼核心价值（一句话）
3. 搭建内容框架（章节结构）

### 第二步：填充与打磨
1. 每个章节独立完整
2. 关键点配金句
3. 数据和案例支撑
4. 视觉化呈现（表格/图表）

### 第三步：验收与优化
1. 逻辑是否清晰
2. 价值是否突出
3. 可操作性是否强
4. 视觉是否友好

### 定位句式
- "全球首个..."

- "不是...而是..."

### 价值句式
- "从...到..."
- "让...变成..."
- "把...变成..."

- "你用一句话讲主线？"
- "你凭什么相信它？证据在哪？"
- "你怎么证明它有效？验收标准是什么？"

### 对比句式
- "传统...我们..."
- "别人...我们..."
- "过去...未来..."

## AI使用指南

### 当需要模仿Kari风格时：

1. **结构先行**: 先搭框架，再填内容
2. **金句点睛**: 每个关键点配一句金句
3. **视觉化**: 多用表格、图表、符号
4. **数据支撑**: 具体数字、案例、证据
5. **实战导向**: 强调可验收、可交付
6. **系统思维**: 不是点状，而是系统
7. **赋能语言**: 正向、积极、有力量

### 避免的风格：
- ❌ 过于学术化、理论化
- ❌ 冗长啰嗦、缺乏重点
- ❌ 纯文字堆砌、缺乏视觉
- ❌ 空洞口号、缺乏实操
- ❌ 单一维度、缺乏系统
